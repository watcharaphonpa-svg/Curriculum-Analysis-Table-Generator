
# คู่มือการใช้งานระบบวิเคราะห์หลักสูตร (Curriculum Matrix Pro)

ระบบนี้ถูกออกแบบมาเพื่อช่วยครูและบุคลากรทางการศึกษาในการวิเคราะห์ค่าน้ำหนักของหลักสูตร เพื่อนำไปสู่การออกข้อสอบที่ได้มาตรฐาน โดยระบบจะแบ่งออกเป็น 3 ตารางหลัก ดังนี้:

## 1. การตั้งค่าข้อมูลเบื้องต้น
- กรอกรายละเอียดวิชาในส่วนบน (รหัสวิชา, ชื่อวิชา, หน่วยกิต ฯลฯ)
- ใช้ปุ่ม **"+ เพิ่มหน่วยการเรียน"** เพื่อเพิ่มแถวใหม่
- ใช้ปุ่ม **"+ เพิ่มพฤติกรรม"** เพื่อเพิ่มคอลัมน์ใหม่ (ค่าเริ่มต้นคือพฤติกรรม 6 ด้านของ Bloom)

## 2. ตารางที่ 1: ตารางวิเคราะห์หลักสูตร (ระดับความสำคัญ)
- **การกรอกข้อมูล**: ให้กรอกตัวเลข **1 - 10** ในแต่ละช่องพฤติกรรม (10 คือสำคัญมากที่สุด)
- **การคำนวณ**: ระบบจะรวมคะแนนในแต่ละแถวและคอลัมน์ให้อัตโนมัติ
- **ลำดับความสำคัญ (Rank)**: ระบบจะจัดอันดับ 1-10 ตามคะแนนรวม (คะแนนมากที่สุดจะได้อันดับ 1)
- **จำนวนคาบ**: กรอกจำนวนชั่วโมงสอนจริงในแต่ละหน่วย

## 3. ตารางที่ 2: น้ำหนักคะแนน (ฐาน 100)
- ตารางนี้จะตัดหัวข้อ **ทักษะพิสัย, จิตพิสัย และประยุกต์ใช้** ออก (แสดงผลเฉพาะด้านพุทธพิสัย)
- ระบบจะคำนวณค่าน้ำหนักโดยนำตัวเลขจากตารางที่ 1 มาปรับสัดส่วนให้ผลรวมทั้งหมดเป็น **100**
- **เงื่อนไข**: ตัวเลขในแต่ละช่องจะถูกจำกัดไม่ให้เกิน **10** (เพื่อความกระจายตัวของข้อสอบ)

## 4. ตารางที่ 3: จำนวนข้อสอบที่ใช้จริง (ฐาน 60)
- ระบบจะนำค่าน้ำหนักจากตารางที่ 2 มาคำนวณเป็นจำนวนข้อสอบจริง โดยอ้างอิงฐาน **60 ข้อ**
- ผลรวมทั้งหมดในตารางนี้ (Items) จะเท่ากับ **60** พอดีเสมอ

## 5. การจัดการปุ่มลบ
- **ลบหน่วยการเรียนรู้**: เอาเมาส์ไปวาง (Hover) ที่แถวหน่วยการเรียนรู้ จะปรากฏไอคอนถังขยะสีแดงที่ด้านขวาของชื่อหน่วย
- **ลบพฤติกรรม**: เอาเมาส์ไปวางที่หัวข้อพฤติกรรม (แนวตั้ง) จะปรากฏปุ่ม (x) สีแดงด้านบน
- **ล้างตัวเลขทั้งหมด**: กดปุ่มสีแดงที่ส่วนควบคุมด้านบนเพื่อล้างตัวเลขในตารางทั้งหมด

## 6. การพิมพ์รายงาน
- กดปุ่ม **"พิมพ์รายงาน"** ที่มุมขวาบน ระบบจะจัดรูปแบบหน้ากระดาษให้เหมาะสมสำหรับการพิมพ์ (ขนาด A4) โดยซ่อนปุ่มควบคุมและส่วนที่ไม่จำเป็นออกให้อัตโนมัติ
